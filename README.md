Interaktive Zählspiel-App

Beschreibung
Dieses Projekt ist ein interaktives Lernspiel, in dem Benutzer zählen müssen, wie viele bestimmte Spielzeuge auf einem Regal stehen.
Die Anwendung zeigt nacheinander verschiedene Fragen mit unterschiedlichen Spielzeugen: Segelboote, Trommeln, Autos, Enten und Eimer.
Eine kleine Spielzeuggrafik unten links hilft dabei, das aktuelle Spielzeug visuell zu erkennen.

Funktionen
Mehrere Fragen mit unterschiedlichen Spielzeugen
Eingabefeld zur Beantwortung der Frage
Überprüfung der Antwort mit sofortigem Feedback
Richtige Antworten werden grün angezeigt, falsche rot
Animierte Anzeige der aktuellen Spielzeugfrage

Technologie
HTML
CSS
JavaScript

Nutzung
Öffnen Sie die index.html Datei im Browser
Zählen Sie die angezeigten Spielzeuge und geben Sie die Zahl ein
Klicken Sie auf Überprüfen
Nach richtiger Antwort wird die nächste Frage angezeigt

Zielgruppe
Kinder, die Zählen lernen
Lernspiele für frühkindliche Bildung
